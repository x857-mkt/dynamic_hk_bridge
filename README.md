# Apple Home 動態標籤橋接器 (Dynamic HomeKit Bridge)

這是一個專為 Home Assistant 設計的自訂整合，允許您將複雜的『實體』動態橋接至 Apple HomeKit。
透過這個套件，您可以更靈活地管理哪些裝置要出現在 Apple 的「家庭」App 中，並隨時調整更新速度，讓整個系統用起來更順手。

---

## 💡 功能特色

* **動態標籤篩選 (Dynamic Tagging)**
  您可以透過標籤 (Tags) 來精確篩選想橋接的裝置。
  > ⚠️ **提醒**：請先在 Home Assistant 中為您的裝置建立好標籤，設定時直接選取即可。
* **彈性調整更新時間**
  系統預設的更新頻率可能不符合您的需求，這裡提供簡單的設定，讓您可以根據裝置特性調整橋接器的同步速度。
* **HACS 友善**
  支援透過 HACS 一鍵安裝與更新，方便日後維護與還原部署。

## 📋 安裝前準備

在安裝本整合之前，請確保您的 Home Assistant 環境已具備以下基礎：

### 1. 基礎環境要求
* **HACS**：本整合透過 HACS 發布，請確保您已完成 [HACS](https://hacs.xyz/) 的安裝與設定。
* **網路環境**：請確保您的 Home Assistant 網路環境正確（建議使用固定 IP，且網路模式設為 **Host**），以利 HomeKit 服務能順利在區域網路中被 iPhone 搜尋與配對。

### 2. 標籤 (Tags) 事先設定（重要！）
本整合核心依賴標籤系統。請在開始設定前，前往 **「設定」➔「分區、標籤與區域」➔「標籤」**，事先建立好您打算用來管理裝置的標籤（例如：`HomeKit`、`Bedroom_Devices` 等），並將標籤套用到目標實體上。

### 3. 環境相容性確認
請根據您目前的 HA 狀態，選擇對應的操作建議：

* **情境 A：如果是全新的 HA 環境**
  您**不需要**手動安裝原生的 HomeKit 整合。本整合啟動時會自動呼叫並配置底層所需的 HomeKit 基礎設施，請直接依循下方「如何安裝」操作。
* **情境 B：如果原本就已有使用原生的 HomeKit 整合**
  為了避免 Apple Home 在同一個網路下發生橋接衝突，建議：
  * **完全移轉**：請先備份現有的 HomeKit 設定，然後移除原有的原生 HomeKit 橋接器，再安裝本整合。
  * **兩者並存**：這也是可行的。但請確保在設定本整合時，**使用與原生整合完全不同的橋接器名稱**，且兩者**切勿重複橋接相同的實體**，以免在 iPhone 「家庭」App 中出現重複裝置或狀態衝突。

## 🚀 如何安裝與配對

1. 進入 Home Assistant 的 **HACS** 頁面。
2. 點擊右上角記號（三個點），選擇 **「自訂儲存庫 (Custom repositories)」**。
3. 在儲存庫網址輸入：`https://github.com/x857-mkt/dynamic_hk_bridge`
4. 類別（Category）選擇 **Integration**，然後點擊 **「新增 (Add)」**。
5. 在列表中找到 **「Apple Home 動態標籤橋接器」**，點擊下載。
6. 下載完成後，**請務必重啟 Home Assistant**。
7. 重啟後，前往 **「設定」➔「裝置與服務」➔「新增整合」**，搜尋 `Apple Home 動態標籤橋接器` 即可開始初始化設定。
8. 設定完成後，系統會自動彈出 **8 位數配對碼 (或 QR Code)**，此時打開 iPhone 的「家庭」App 掃描或輸入代碼，即可完成與 Apple Home 的連線！
